# Golang SDK

## 简介
此 SDK 适用于 Golang 1.16 及以上版本。

使用此 SDK 可以构建您的云湖机器人，能让您以非常便捷的方式和云湖服务进行交互。


## 开源
https://github.com/yhchat/bot-go-sdk

## 功能列表
* 消息订阅功能
* 消息发送功能
* 批量消息发送功能
