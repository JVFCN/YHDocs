## 云湖用户 Daen 大佬编写的 Java版本SDK

    云湖注: 该SDK质量和完善度很高，推荐使用。

云湖SDK Core服务，引入此 Core 可以快速构建您的云湖机器人，能让您以非常便捷的方式和云湖服务进行交互。

Core基于SpringBoot2.x、jdk8开发（您可以根据需求进行更换）
